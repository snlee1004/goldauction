# 게시판 관리자 모드 기능 명세서

## 📋 개요
게시판을 생성하고 관리할 수 있는 관리자 모드 시스템입니다. 일반 게시판과 이벤트 공동구매 게시판 두 가지 타입을 지원하며, 각 게시판마다 공지사항 관리 기능을 제공합니다.

---

## 🎯 주요 기능

### 1. 게시판 생성 기능

#### 1.1 게시판 타입 선택
- **일반 게시판**: 기본적인 커뮤니티 게시판
- **이벤트 공동구매 게시판**: 쇼핑몰 형태의 상품 판매 게시판

#### 1.2 게시판 생성 시 입력 항목
- 게시판명
- 게시판 설명
- 게시판 타입 (일반/이벤트 공동구매)
- 게시판 카테고리 (선택사항)
- 활성화 여부

---

## 📝 1. 일반 게시판 기능

### 1.1 글 작성 권한
- **작성 가능**: 가입된 회원만 글 작성 가능
- **수정/삭제**: 작성자 본인 또는 관리자만 가능
- **조회**: 비회원도 게시글 내용 조회 가능

### 1.2 비속어 검열 기능
- 게시글 제목 및 내용에 비속어 필터링
- 비속어 발견 시 작성 불가 또는 자동 마스킹 처리
- 관리자 설정 가능한 비속어 목록 관리

### 1.3 댓글 기능
- **댓글 작성**: 가입된 회원만 댓글 작성 가능
- **댓글 수정/삭제**: 작성자 본인 또는 관리자만 가능
- **댓글 조회**: 비회원도 댓글 조회 가능
- 대댓글 기능 (선택사항)

### 1.4 게시판 기능
- 게시글 목록 조회 (페이지네이션)
- 게시글 검색 (제목, 내용, 작성자)
- 게시글 정렬 (최신순, 조회수순, 댓글순)
- 첨부파일 업로드 (이미지, 문서 등)
- 게시글 좋아요/추천 기능 (선택사항)

---

## 🛒 2. 이벤트 공동구매/특가구매 게시판

### 2.1 게시판 형태
- 일반적인 쇼핑몰 상세페이지 형태로 구성
- 상품 이미지, 상세 설명, 가격 정보 표시
- 옵션 선택 기능 (색상, 사이즈 등)

### 2.2 공동구매/특가판매 기능
- **종료일 설정**: 이벤트 종료일시 설정
- **재고 관리**: 상품 재고 수량 관리
- **자동 마감 처리**: 
  - 종료일시 도달 시 자동 마감
  - 재고 소진 시 자동 마감
- **주문 기능**: 회원 주문 및 결제 연동 (선택사항)

### 2.3 상품 정보 관리
- 상품명, 상품 설명
- 상품 가격 (정가, 할인가)
- 상품 이미지 (다중 이미지 지원)
- 상품 옵션 (색상, 사이즈 등)
- 재고 수량
- 배송 정보

### 2.4 이벤트 상태 관리
- **진행중**: 이벤트 진행 중
- **마감**: 종료일 도달 또는 재고 소진
- **종료**: 이벤트 종료
- 상태별 필터링 및 표시

---

## 📢 3. 공지사항 관리 기능

### 3.1 공지사항 작성 권한
- **작성/수정/삭제**: 관리자만 가능
- **조회**: 모든 사용자 (회원/비회원) 조회 가능

### 3.2 공지사항 기능
- 각 게시판마다 독립적인 공지사항 관리
- 공지사항 상단 고정 기능
- **상단 노출 줄수 설정**: 관리자가 상단에 노출할 공지사항 개수 설정
- 공지사항 우선순위 설정 (선택사항)

### 3.3 공지사항 표시
- 게시판 목록 상단에 공지사항 영역 표시
- 공지사항은 일반 게시글과 구분하여 표시
- 공지사항 아이콘 또는 배지 표시

---

## 🔧 관리자 기능

### 4.1 게시판 관리
- 게시판 생성/수정/삭제
- 게시판 활성화/비활성화
- 게시판 순서 조정

### 4.2 게시글 관리
- 게시글 삭제
- 게시글 공지사항으로 전환
- 게시글 숨김 처리
- 게시글 신고 관리

### 4.3 댓글 관리
- 댓글 삭제
- 댓글 숨김 처리
- 댓글 신고 관리

### 4.4 비속어 관리
- 비속어 목록 추가/수정/삭제
- 비속어 검열 설정 (자동 마스킹/작성 불가)

## 📊 데이터베이스 설계 (초안)

### 게시판 테이블 (BOARD)
- board_seq (게시판 번호)
- board_name (게시판명)
- board_description (게시판 설명)
- board_type (게시판 타입: 일반/이벤트)
- is_active (활성화 여부)
- created_date (생성일)
- updated_date (수정일)

### 게시글 테이블 (BOARD_POST)
- post_seq (게시글 번호)
- board_seq (게시판 번호)
- member_seq (작성자 번호)
- post_title (제목)
- post_content (내용)
- is_notice (공지사항 여부)
- notice_order (공지사항 순서)
- view_count (조회수)
- created_date (작성일)
- updated_date (수정일)

### 댓글 테이블 (BOARD_COMMENT)
- comment_seq (댓글 번호)
- post_seq (게시글 번호)
- member_seq (작성자 번호)
- comment_content (댓글 내용)
- parent_comment_seq (부모 댓글 번호 - 대댓글용)
- created_date (작성일)
- updated_date (수정일)

### 이벤트 상품 테이블 (EVENT_PRODUCT)
- product_seq (상품 번호)
- board_seq (게시판 번호)
- product_name (상품명)
- product_description (상품 설명)
- original_price (정가)
- sale_price (할인가)
- stock_quantity (재고 수량)
- end_date (종료일시)
- event_status (이벤트 상태: 진행중/마감/종료)
- created_date (생성일)
- updated_date (수정일)

### 비속어 테이블 (PROFANITY_FILTER)
- filter_seq (필터 번호)
- profanity_word (비속어)
- replacement_word (대체어 - 선택사항)
- is_active (활성화 여부)

## 🎨 UI/UX 요구사항

### 게시판 목록 페이지
- 게시판별 탭 또는 메뉴 구성
- 공지사항 상단 고정 영역
- 게시글 목록 (제목, 작성자, 작성일, 조회수)

- 검색 기능

- 댓글 영역
- 작성자 정보
- 수정/삭제 버튼 (권한 있는 경우)

### 이벤트 게시판 상세 페이지
- 상품 이미지 갤러리
- 상품 정보 (가격, 옵션, 재고)
- 주문/구매 버튼
- 상품 상세 설명
- 이벤트 종료일시 표시

### 관리자 페이지
- 게시판 관리 대시보드
- 게시판 생성/수정 폼
- 게시글 관리 목록
- 통계 정보 (게시글 수, 댓글 수 등)

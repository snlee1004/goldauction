# 골드옥션 시스템 발표 자료
## 6~10분 발표용

---

## 1. 프로젝트 개요

### 골드옥션 (Gold Auction)
- **프로젝트명**: 골드옥션
- **목적**: 골드 및 귀금속 경매 플랫폼
- **주요 기능**: 경매, 입찰, 게시판, 공동구매, 관리자 시스템

### 기술 스택
- **Frontend**: React + Vite
- **Backend**: Spring Boot 3 (Java 17+)
- **Database**: Oracle DB
- **Build Tool**: Gradle

---

## 2. 시스템 아키텍처

### 3-Tier Architecture
```
[Frontend: React]
    ↕ HTTP/REST API
[Backend: Spring Boot]
    ↕ JDBC/JPA
[Database: Oracle]
```

### 주요 컴포넌트
- **Frontend**: React 컴포넌트 기반 UI
- **Backend**: RESTful API 서버
- **Database**: Oracle DB (19개 테이블)

---

## 3. 핵심 기능 - 경매 시스템

### 경매 상품 관리
- 상품 등록 (다중 이미지 지원)

- 경매 기간 설정 (7일, 14일, 21일, 30일)
- 거래 방식 선택 (직거래, 매장방문, 에스크로, 중계소)

### 입찰 시스템

- 최고 입찰 금액 표시
- 입찰 이력 조회

- 즉시 구매 옵션

## 8. 데이터베이스 구조

### 주요 테이블 (19개)
- **회원 관련**: MEMBER1, MANAGER1
- **경매 관련**: IMAGEBOARD1, IMAGEBOARD_IMAGES1, BID1
- **게시판 관련**: BOARD1, BOARD_POST1, BOARD_COMMENT1, BOARD_POST_FILE1
- **이벤트 관련**: EVENT_PRODUCT1, EVENT_PRODUCT_IMAGE1, EVENT_PRODUCT_OPTION1, EVENT_ORDER1
- **관리 관련**: CSS_SET1, CSS_FILE1, CSS_APPLY1, PROFANITY_FILTER1, BOARD_NOTICE_SETTING1, BOARD_NOTIFICATION1
- **기타**: NOTICE1, POPUP1

### 특징
- 외래키 제약조건으로 데이터 무결성 보장
- 인덱스 최적화로 성능 향상
- 시퀀스 기반 PK 관리

## 9. 주요 화면 구성

### 사용자 화면
- **메인페이지**: 차트, 공지사항, 카테고리별 상품 목록
- **경매 목록**: 카테고리별, 상태별 필터링
- **경매 상세**: 상품 정보, 입찰 내역, 입찰하기
- **게시판**: 게시글 목록, 상세보기, 댓글

### 관리자 화면
- **대시보드**: 통계 차트, 주요 지표
- **회원 관리**: 회원 목록, 정보 수정, 정지 관리
- **게시판 관리**: 게시판/게시글/댓글 관리
- **경매 상품 관리**: 상품 목록, 수정/삭제, 판매 관리
- **통계**: 거래 통계, 카테고리별 통계, 시간대별 현황

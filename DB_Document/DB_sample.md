# 샘플 데이터 입력 쿼리 모음

## 📋 개요
새로운 PC에 DB를 셋팅한 후 테스트용 샘플 데이터를 입력하기 위한 쿼리 모음입니다.

**⚠️ 주의사항**: 
- 이 파일의 쿼리들은 **DB.md**와 **DB_board.md**의 테이블 생성이 완료된 후 실행해야 합니다.
- 외래키 제약조건을 고려하여 순서대로 실행하세요.
- 실제 운영 환경에서는 이 샘플 데이터를 사용하지 마세요.
